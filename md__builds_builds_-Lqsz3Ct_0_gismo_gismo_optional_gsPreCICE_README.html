Start here

  1. Install preCICE.
  2. Clone the preCICE tutorials repository.*
  3. Get G+Smo and the adapter.

The gsPreCICE module is a submodule of the G+Smo library. Follow the steps below to download and configure it: ``` git clone https://github.com/gismo/gismo.git cd gismo mkdir build cd build cmake .. -DGISMO_OPTIONAL="<Other submodules>;gsPreCICE" ```

  1. Build examples. ``` make <tutorial file="" name>=""> -j <number_of_threads> ```
  2. Link the compiled executable to the gismo-executable folder within the tutorial directory.* ``` cd <Your precice="" tutorial="" folder>="">/partitioned-heat-conduction/gismo-executable ln -sf <You G+Smo build folder>/bin/<executable name>=""> ./gismo_executable ``
  3. Open two terminals and run. ``` cd solid-gismo ./run.sh ```

``` cd fluid-<other solvers>=""> ./run.sh ```

Note: You need to perform steps 2 and 5 if you want to run the simulation with other libraries.
