◆ applyPCDboundaryConditions()

template<class T >
void applyPCDboundaryConditions ( gsSparseMatrix< T > & Ap,
gsSparseMatrix< T > & Fp,
int bcType )

Apply boundary modifications to the pressure Poisson and convection matrices.

This is needed for the PCD block preconditioner.

Admissible values of bcType and their meaning: 0 - Dirichlet inflow + Neumann outflow and walls for both \( Ap, Fp \) 1 - Robin inflow + Neumann outflow and walls for \( Fp \), nothing for \( Ap\) ( \( Ap \) should not be assembled, i.e., \( B \hat{M}_u^{-1} B^T \ \)) 2 - Robin inflow + Dirichlet outflow + Neumann walls for \( Fp \), nothing for \( Ap \) ( \( Ap\) should not be assembled) 3 - Robin inflow + Dirichlet outflow + Neumann walls for \( Fp \), Dirichlet outflow + Neumann inflow and walls for \( Ap \) ( \( Ap \) should be assembled) 4 - no boundary modification

Parameters
[in,out]Apa reference to the pressure Poisson matrix to be modified
[in,out]Fpa reference to the pressure convection matrix to be modified
[in]bcTypeboundary conditions strategy

◆ assembleBlockNpattern()

template<class T >
void assembleBlockNpattern ( )

Assemble block N (multiplied by 0) to get its sparsity pattern.

Since block N arises from the nonlinear convective term, it is reassembled in each step of the linearization method. The block assembled here is used for exact memory allocation in the global matrix.

◆ getPressurePoissonMatrix()

template<class T >
gsSparseMatrix< T > getPressurePoissonMatrix ( bool assemb = true,
bool lumping = false )

Returns the Poisson matrix for the pressure basis.

This is needed for the PCD block preconditioner.

The matrix \( A_p \) is either assembled or defined as \( B \hat{M}_u^{-1} B^T \ \), where \( \hat{M}_u_\) is an approximation of the velocity mass matrix.

The function assemblePressurePoisson() is called here, if was not called before.

Parameters
[in]assembassemble the matrix
[in]lumpinguse mass lumping (in the case assemb = false)
