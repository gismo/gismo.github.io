Detailed Description

template<class T>
class gismo::MultiWidthMatrix< T >

A class to manage a collection of matrices with varying row sizes.

Template Parameters
TThe type of the elements in the matrices.

This class allows for the storage and management of multiple matrices that may have different row sizes. It provides functionality to add new matrices and access rows from the combined collection.
