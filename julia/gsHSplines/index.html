HBSpline(basis::Basis, coefs::Matrix{Cdouble})

Arguments

  • basis::Basis: a basis object
  • coefs::Matrix{Cdouble}: a matrix of coefficients

Examples

b = BSplineBasis(kv)
hb = HBSplineBasis(b)
